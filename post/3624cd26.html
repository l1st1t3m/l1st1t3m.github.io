VPS系列01：AWS账号的注册流程

发表于 2026-02-18 分类于 VPS系列

本文详细介绍如何创建 Amazon Web Services (AWS) 账户创建指南，从注册流程、绑定信用卡验证，到启用双重验证（MFA）和 IAM 安全配置的全流程操作方法，适合首次使用 AWS 云服务的个人用户。

创建亚马逊AWS账号、绑定信用卡、账号安全设置

需求：VISA或双币信用卡 💳 ，境内外均可。
中国境内银行的外币信用卡大多数会与国内信用卡捆绑注册，可以通过申请后再与开户行申请注销国内信用卡，只保留外币信用卡。（免年费的银行首选）

创建亚马逊账户

亚马逊AWS官方网站：AWS
进入亚马逊AWS官方网站，选择右上角创建AWS账户，如下填写注册邮件地址、密码、账户名，然后点击继续。


按以下标注信息填写，最好与申请信用卡时的申请信息一致，以便后期审核通过，然后点击继续。
注：全名与地址需英文填写，在线翻译即可。在线翻译_有道
例:
宝山区示范新村37号403室
(Room 403, No. 37, SiFang Residential Quarter, BaoShan District )
河南省南阳市中州路42号
(Zhongzhou Road, Nanyang City Henan Prov. China 473004)

绑定信用卡

AWS验证信用卡，填写信用卡相关信息，通过后会从所绑定的外币信用卡中扣除1$来验证卡的可用性，稍后会
退还，验证成功后才可继续。

验证手机号，使用信用卡预留手机号即可。

验证时常在30min-24h内不等，此期间可关注注册邮箱内亚马逊官方发来的邮件，直到可正常登录官网。

账号基础配置

创建完毕AWS账户后，请进入亚马逊AWS的IAM服务界面。

首先开启MFA双重验证，增强登陆的安全性；
其次如果开启了IAM，请停用或删除IAM帐户的访问密钥，除非您能搞清楚权限赋予，确保账号安全。

这样就完成了亚马逊AWS账号的注册。
